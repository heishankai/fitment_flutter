# 装修APP

1. 创建工程： flutter create domo

2. 启动 ios 模拟器：xcrun simctl boot "iPhone 15 Pro (0237DD1A-46FD-4BB7-B09E-D4518A8C1779)" && open -a Simulator

3. 启动 android 模拟器 : $ANDROID_HOME/emulator/emulator -avd Pixel_9_Pro

4. 启动项目： flutter runclear

5. 只创建 ios 和 安卓项目 ：flutter create --platforms android,ios fitment_flutter
